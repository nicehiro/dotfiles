# .config

Application configuration files, symlinked to `~/.config/` via Stow.

## Contents

| Directory | Description |
|-----------|-------------|
| `nvim/` | Neovim config — Lazy.nvim plugin manager, Lua-based (`init.lua` → `lua/config/`) |
| `ghostty/` | Ghostty terminal — key-value config |
| `tmux/` | Tmux — vi-mode, vim-style pane nav, TPM plugins |
| `alacritty/` | Alacritty terminal — TOML config with Tokyo Night theme |
| `starship.toml` | Starship prompt — nerd font symbols |
| `rime/` | Rime input method (Chinese) — double pinyin with custom dictionaries |
| `emacs-rime/` | Emacs Rime integration |
| `gh/` | GitHub CLI config |
| `git/` | Global gitignore |
| `linearmouse/` | LinearMouse settings |
| `mole/` | Mole config |
| `opencode/` | OpenCode AI agent — config, themes, and academic prompt templates |
| `zellij/` | Zellij terminal multiplexer config |
| `Code/` | VS Code custom CSS |
